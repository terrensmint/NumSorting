# Основы информатики - Сортировка чисел (Bash)

## Задача
Написать скрипт для сортировки заданного пользователем набора чисел.
Скрипт принимает числа в качестве аргументов командной строки.
Необходимо реализовать функцию для сортировки массива.
Отсортировать массив и вывести результат на экран.

## Установка
```bash
# Клонировать репозиторий
git clone git@github.com:terrensmint/NumSorting.git

# Сделать скрипт исполняемым
chmod +x script.sh
```

## Использование
В командной строке перечислить целые числа для сортировки после наименования скрипта, например:
```bash
./script.sh 5 2 6 19 13 42 13 4
```
